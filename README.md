# BME-450---project
## Title:
Bird Identification Based On Sounds
## Team Members:
Ainsley Gillcrist (agillcri), Akansha Chauhan (akanshachau)

### Description:  
Bird watching is a common activity throughout Northern America. Many birdwatchers identify birds based on appearance; however, identification based on their songs is a neat way to identify birds that one may not be able to see. Each bird species has its own song, and due to the prevalence of bird watching, there are multiple datasets of bird songs online. In order to narrow the scope of the project, we will focus on specific owl species, specifically the Barred Owl, the American Barn Owl, and the Great Horned Owl. The audio files we are using are from Xeno-Canto, a website with a collection of many different noises from species throughout the world. Using a dataset from this source can help ensure the accuracy of the bird noises. Overall, the goal of this project is to create a neural network that can identify three different owls based on their calls.
